This page allows you to generate and share dynamic vector layers built on top of OpenStreetMap (OSM). Each object shown on the map has a series of tags associated with it, using the rules tab, you can define rules to select a subset of those objects.

Combine one or more rules to create a custom layer, after your layer has been submitted, you'll receive a unique ID for the layer that you can use to share your map.

https://tiles.maphub.dev/index.html?id=${your_layer_id}
